Ejercicio 1

  1. Se abre un prompt que solicita tu nombre (debe ser sin numeros, solo texto)
  2. Después se ejecuta un prompt solicitando tu edad
  3. Si la edad es menor a 18, se abrirá un alert diciendo cuantos años le faltan para poder ingresar
  4. Si la edad es mayor a 18, se abrirá un prompt pidiendo ingresar el nombre
  5. Se ejecturá un alert dando la bienvenida y mostrando el nombre ingresado
